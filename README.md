# python-api-challenge
Module 6 Challenge

This python-api challenge project has two parts one used an API request, and Python script to visualize the weather date of 600+ cities across the world, and its under WeatherPy folder. The cities dataset found in output_data folder which includes the following columns City_ID,City,Cloudines,Country,Date, Humidity,Lat,Lng,Max,Temp,Wind Speed. The data analyzed, and displayed on WeatherPy jupyter notebook.
The other part of the challenge VacationPy is used the output data, and investigate the data to plan for future vacations, and to select hotels. The analysis covers humidity heatmap in the cities that sourced from the weather data, and spot the hotels on top of the humidity heatmap. We can look at the map images in the output folder for both the heatmap and the hotel heatmap images.
